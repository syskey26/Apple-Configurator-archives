About the device browser

The device browser is the main window that appears when you launch Apple Configurator. It contains the following:

  • Favorites bar: Shows the four default groups:

    • All devices: All connected devices

    • Supervised: All supervised connected devices

    • Unsupervised: All unsupervised connected devices

    • Recovery: All devices currently in recovery mode

  • Search bar: You can search for specific devices or tags and save the searches as new favorites

  • Default toolbar: You can customize the toolbar by choosing View > Customize toolbar

In the device browser, you can also:

  • Prepare a device, so it has the mobile device management (MDM) information and supervision identity attached

  • Add specific actions to a device, such as putting it in Single App Mode

  • Double-click the device to view the device information

To learn more about what Apple Configurator can do, see the Apple Configurator overview.
